# DevGigs 🚀

> The #1 Freelance Job Board for Developers

A full-stack SaaS platform where companies post freelance projects and developers find work. Built with Next.js 15, PostgreSQL, Stripe, and NextAuth.

## ✨ Features

- **Browse Jobs** — Filter by type (Remote / Onsite / Hybrid) and skills
- **Post a Job** — Companies pay $29 to list a freelance project
- **Google Auth** — One-click sign in with Google
- **Stripe Payments** — Secure one-time payment to activate job listing
- **Apply System** — Developers apply with one click
- **Company Dashboard** — View all posted jobs and applicants
